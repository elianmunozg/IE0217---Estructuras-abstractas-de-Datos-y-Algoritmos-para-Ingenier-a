Detailed Description

Clase para representar un préstamo.

La clase Prestamo permite modelar diferentes tipos de préstamos y realizar operaciones relacionadas con los mismos, como el cálculo de la cuota mensual, abonar cuotas, generar reportes y generar tablas de amortización.
