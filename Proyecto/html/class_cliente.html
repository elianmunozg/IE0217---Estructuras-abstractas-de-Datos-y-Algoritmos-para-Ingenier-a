template<typename T >
void Cliente< T >::agregarCuenta ( const std::string & tipoCuenta)

Agrega una nueva cuenta al cliente.

Agrega una nueva cuenta bancaria al cliente con un número de cuenta generado aleatoriamente.

Parameters
tipoCuentaEl tipo de cuenta a agregar (por ejemplo, "Ahorros").

Este método asigna un nuevo número de cuenta, generado aleatoriamente, a un tipo específico de cuenta bancaria (como "Dolares" o "Colones"). Esto permite al cliente tener múltiples cuentas de diferentes tipos, cada una identificada por un número único.

Parameters
tipoCuentaEl tipo de la nueva cuenta a agregar, especificando su categoría (por ejemplo, "Colones").
